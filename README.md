neovim-buckler
==============

This Neovim plugin maintains a linear yank history and automatically
synchronizes it with the unnamed, numbered and clipboard registers.

See also [neovim-cutlass](https://github.com/Coacher/neovim-cutlass).

Related plugins
===============

* [vim9-buckler](https://github.com/Coacher/vim9-buckler) for Vim9.
  Actively maintained.

* [vim-yoink](https://github.com/svermeulen/vim-yoink) for Vim 8.
  Unmaintained.

* [vim-easyclip](https://github.com/svermeulen/vim-easyclip) for Vim 7.
  Unmaintained.
